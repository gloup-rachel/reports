🔵 미리보기

아래 설정을 수정한 뒤 상단 미리보기 버튼을 클릭하면, 현재 설정이 반영된 대시보드를 페이지 하단에서 바로 확인할 수 있습니다. 실제 배포 전에 데이터가 올바르게 표시되는지 점검하는 용도입니다.

🟢 index.html 다운로드

현재 입력된 설정(리포트 제목, 기간, 인사이트 등)이 주입된 정적 대시보드 파일을 다운로드합니다. 아래 순서로 사용하세요.

  1. 1Admin에서 해당 월의 설정을 모두 입력
  2. 2index.html 다운로드 클릭 → 파일이 로컬에 저장됨
  3. 3GitHub 저장소(gloup-rachel/reports)에 접속
  4. 4기존 index.html 파일을 다운로드받은 파일로 교체 업로드
  5. 51~2분 후 GitHub Pages에 반영 완료

※ live.html은 실시간 데이터를 자동으로 불러오므로 별도 업로드가 필요하지 않습니다. index.html만 교체하면 됩니다.

⬜ 설정 JSON 내보내기

현재 Admin에 입력된 모든 설정값을 .json 파일로 저장합니다. 매월 설정을 백업하거나, 팀원과 설정을 공유할 때 사용하세요.

  • 파일명 형식: respace_settings_2026-03.json
  • 리포트 정보, 주차 설정, 인사이트, 액션 아이템 등 모든 항목이 포함됩니다

⬜ 설정 JSON 불러오기

이전에 내보낸 .json 파일을 불러와 Admin 폼에 자동으로 채워넣습니다. 활용 시나리오:

  • 월간 교체 — 지난달 JSON을 불러온 뒤 월/기간만 수정하면 빠르게 세팅 가능
  • 백업 복원 — 실수로 내용을 지웠을 때 직전 백업 JSON을 불러오기
  • 팀 공유 — 다른 담당자가 세팅한 JSON을 받아서 불러오기
